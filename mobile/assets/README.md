# Assets Directory

Please add the following required assets:

- `icon.png` - App icon (1024x1024 PNG)
- `splash.png` - Splash screen image (1284x2778 PNG for iPhone 12 Pro Max)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024 PNG)
- `favicon.png` - Web favicon (48x48 PNG)

You can generate these assets using:
- `expo install expo-splash-screen` and then `npx expo install --fix`
- Or use online tools like https://appicon.co/

For now, you can use placeholder images or the existing assets from your assets folder.
